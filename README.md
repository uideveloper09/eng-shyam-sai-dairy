# Engineer Shyam Sai Dairy

Public website for **Engineer Shyam Sai Dairy** — fresh milk, dahi, ghee, paneer, butter, and related products, with emphasis on daily delivery and product information.

## Features

- Product showcase and branding pages
- Responsive layout (mobile-friendly)
- Hindi-friendly content where applicable

## Tech stack

- [React](https://react.dev/) 18
- [Vite](https://vitejs.dev/) 5
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/) (Radix primitives)
- [React Router](https://reactrouter.com/) for routing
- [TanStack Query](https://tanstack.com/query) for data fetching patterns (where used)

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- npm (comes with Node)

## Getting started

```bash
git clone https://github.com/uideveloper09/eng-shyam-sai-dairy.git
cd eng-shyam-sai-dairy
npm install
npm run dev
```

The dev server runs at **http://localhost:8080** (see `vite.config.ts`).

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start dev server (Vite)  |
| `npm run build`   | Production build → `dist/` |
| `npm run preview` | Preview production build locally |
| `npm run lint`    | Run ESLint               |
| `npm test`        | Run Vitest (unit tests)  |

## Build output

`npm run build` generates static files in `dist/`, suitable for any static host (GitHub Pages, Netlify, Vercel, S3, etc.).
